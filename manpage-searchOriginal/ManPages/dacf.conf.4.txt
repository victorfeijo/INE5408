dacf.conf <-> device auto-configuration configuration file
/etc/dacf.conf
The kernel uses the dacf.conf file to automatically configure hot plugged devices. Because the dacf.conf file con<hy>
tains important kernel state information, it should not be modified.
The format of the /etc/dacf.conf file is not public and might change in versions of the Solaris operating environment
that are not compatible with Solaris 8.
