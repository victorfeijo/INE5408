mlib_SignalWhiteNoise_F32 <-> white noise generation
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_SignalWhiteNoise_F32(mlib_f32 *wnoise,
 void *state, mlib_s32 n);
The mlib_SignalWhiteNoise_F32() function generates one packet of white noise and updates the internal state.
The function takes the following arguments:
wnoise Generated white noise array.
state Internal state structure.
n Length of the generated sine wave array in number of samples.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
