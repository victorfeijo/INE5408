mlib_ImageConstMul_Fp_Inp <-> multiply with a constant
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageConstMul_Fp_Inp(mlib_image *srcdst,
 const mlib_d64 *c);
The mlib_ImageConstMul_Fp_Inp() function multiplies each pixel in a floating-point image by a constant value
on a pixel-by-pixel basis, in place.
It uses the following equation:
 srcdst[x][y][i] = c[i] * srcdst[x][y][i]
The function takes the following arguments:
srcdst Pointer to source and destination image.
c Constant by which each pixel is multiplied. c[i] contains the constant for channel i.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
