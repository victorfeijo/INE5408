mlib_SignalMulRectangular_F32, mlib_SignalMulRectangular_F32S <-> rectangular windowing multiplication
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_SignalMulRectangular_F32(mlib_f32 *srcdst, mlib_s32 m,
 mlib_s32 n);
mlib_status mlib_SignalMulRectangular_F32S(mlib_f32 *srcdst, mlib_s32 m,
 mlib_s32 n);
Each of these functions performs multiplication of the rectangular window.
Each of the functions takes the following arguments:
srcdst Input and output signal array.
m Rectangular window parameter.
n Number of samples in signal and window arrays.
Each of the functions returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
