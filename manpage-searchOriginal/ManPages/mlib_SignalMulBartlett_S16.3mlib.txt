mlib_SignalMulBartlett_S16, mlib_SignalMulBartlett_S16S <-> Bartlett windowing multiplication
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_SignalMulBartlett_S16(mlib_s16 *srcdst, mlib_s32 n);
mlib_status mlib_SignalMulBartlett_S16S(mlib_s16 *srcdst, mlib_s32 n);
Each of these functions performs multiplication of the Bartlett window.
Each of the functions takes the following arguments:
srcdst Input and output signal array.
n Number of samples in signal and window arrays.
Each of the functions returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
