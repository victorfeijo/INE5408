mlib_ImageLog_Fp <-> computes the natural logarithm of the image pixels
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageLog_Fp(mlib_image *dst, const mlib_image *src);
The mlib_ImageLog_Fp() function computes the natural logarithm of the image pixels.
It uses the following equation:
 dst[x][y][i] = log(src[x][y][i])
The function takes the following arguments:
dst Pointer to destination image.
src Pointer to source image.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
