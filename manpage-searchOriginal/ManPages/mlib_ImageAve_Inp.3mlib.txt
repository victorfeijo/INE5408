mlib_ImageAve_Inp <-> average of two images, in place
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageAve_Inp(mlib_image *src1dst, const mlib_image *src2);
The mlib_ImageAve_Inp() function computes the average of two images on a pixel-by-pixel basis, in place.
It uses the following equation:
 src1dst[x][y][i] = (src1dst[x][y][i] + src2[x][y][i] + 1) / 2
The function takes the following arguments:
src1dst Pointer to first source and destination image.
src2 Pointer to second source image.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
