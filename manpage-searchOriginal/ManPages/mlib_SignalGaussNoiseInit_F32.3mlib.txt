mlib_SignalGaussNoiseInit_F32 <-> Gaussian noise generation
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_SignalGaussNoiseInit_F32(void **state, mlib_f32 mag,
 mlib_f32 mean, mlib_f32 stddev, mlib_f32 seed);
The mlib_SignalGaussNoiseInit_F32() function allocates memory for an internal state structure and converts the
parameters into an internal representation.
The function takes the following arguments:
state Internal state structure.
mag Magnitude of the Gaussian noise to be generated, in Q15 format.
mean Mean of the Gaussian noise.
stddev Standard deviation of the Gaussian noise.
seed Seed value for the pseudorandom number generator.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
