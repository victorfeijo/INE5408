mlib_ImageMax_Fp_Inp <-> two image maximum
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageMax_Fp_Inp(mlib_image *src1dst, const mlib_image *src2);
The mlib_ImageMax_Fp_Inp() function accepts input from the two floating-point source images and writes the
maximum, in place, on a pixel-by-pixel basis.
It uses the following equation:
 src1dst[x][y][i] = MAX{ src1dst[x][y][i], src2[x][y][i] }
The function takes the following arguments:
src1dst Pointer to source and destination image.
src2 Pointer to second source image.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
