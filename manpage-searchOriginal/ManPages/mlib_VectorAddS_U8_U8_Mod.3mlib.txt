Each of these functions adds a scalar to a vector.
For real data, the following equation is used:
 z[i] = c[0] + x[i]
where i = 0, 1, ..., (n - 1).
For complex data, the following equation is used:
 z[2*i] = c[0] + x[2*i]
 z[2*i + 1] = c[1] + x[2*i + 1]
where i = 0, 1, ..., (n - 1).
Each of the functions takes the following arguments:
z Pointer to the first element of the destination vector.
x Pointer to the first element of the source vector.
c Pointer to the source scalar. When the function is used with complex data types, c[0] contains the scalar for the real
part, and c[1] contains the scalar for the imaginary part.
n Number of elements in the vectors.
Each of the functions returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
