mlib_ImageSub1_Inp <-> subtraction, in place
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageSub1_Inp(mlib_image *src1dst,
 const mlib_image *src2);
The mlib_ImageSub1_Inp() function subtracts the second source image from the first source image on a pixel-by-
pixel basis, in place.
It uses the following equation:
 src1dst[x][y][i] = src1dst[x][y][i] - src2[x][y][i]
The function takes the following arguments:
src1dst Pointer to first source and destination image.
src2 Pointer to second source image.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
