mlib_ImageConstNotAnd <-> NotAnd with a constant
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageConstNotAnd(mlib_image *dst, const mlib_image *src,
 const mlib_s32 *c);
The mlib_ImageConstNotAnd() function computes the logical And of the source image with a constant and then
takes the logical Not of that result on a pixel-by-pixel basis.
It uses the following equation:
 dst[x][y][i] = ~(c[i] & src[x][y][i])
The data type of the images can be MLIB_BIT, MLIB_BYTE, MLIB_SHORT, MLIB_USHORT, or
MLIB_INT.
The function takes the following arguments:
dst Pointer to destination image.
src Pointer to source image.
c Array of constants to be applied to each pixel. c[i] contains the constant for channel i.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
