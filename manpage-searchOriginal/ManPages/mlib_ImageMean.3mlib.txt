mlib_ImageMean <-> image mean
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageMean(mlib_d64 *mean, const mlib_image *img);
The mlib_ImageMean() function computes the mean value of all the pixels in the image.
It uses the following equation:
  1 w-1 h-1
 mean[i] = ----- * SUM SUM img[x][y][i]
  w*h x=0 y=0
The function takes the following arguments:
mean Pointer to mean array, where length is the number of channels in the image. mean[i] contains the mean of
channel i.
img Pointer to an image.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
