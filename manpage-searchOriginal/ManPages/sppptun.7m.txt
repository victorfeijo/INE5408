sppptun <-> PPP tunneling pseudo-driver
/dev/sppptun
The /dev/sppptun pseudo-driver provides an interface for tunneling PPP sessions. This interface provides PPP over
Ethernet (PPPoE) service with Solaris PPP.
/dev/sppptun Solaris PPP tunneling device driver.
