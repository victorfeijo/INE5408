pcmem <-> PCMCIA memory card nexus driver
The pcmem driver identifies the type of memory card in the system and will allow future support of other memory
device types.
The PCMCIA memory card nexus driver supports PCMCIA memory card client drivers. There are no user-config<hy>
urable options for this driver.
