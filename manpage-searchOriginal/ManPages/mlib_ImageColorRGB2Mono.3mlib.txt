mlib_ImageColorRGB2Mono <-> RGB to monochrome conversion
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageColorRGB2Mono(mlib_image *dst, const mlib_image *src,
 const mlib_d64 *weight);
The mlib_ImageColorRGB2Mono() function performs a conversion from a red/green/blue to a monochromatic im<hy>
age. The source image must be a three-channel image. The destination image must be a single-channel image.
It uses the following equation:
 dst[x][y][0] = weight[0]*src[x][y][0] +
  weight[1]*src[x][y][1] +
  weight[2]*src[x][y][2]
The function takes the following arguments:
dst Pointer to destination image.
src Pointer to source image.
weight Array of three blending coefficients. It is recommended that these sum to 1.0, but it is not required.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
