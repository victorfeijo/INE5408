mlib_SignalGenHanning_S16 <-> Hanning window generation
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_SignalGenHanning_S16(mlib_s16 *window, mlib_s32 n);
The mlib_SignalGenHanning_S16() function generates the normalized coefficients of the Hanning window.
The function takes the following arguments:
window Generated window coefficient array. The window coefficients are in Q15 format.
n Length of window array.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
