mlib_ImageYProj_Fp <-> image Y projection
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_ImageYProj_Fp(mlib_d64 *yproj, const mlib_image *img);
The mlib_ImageYProj_Fp() function computes the sum of the pixels in each row of the floating-point source im<hy>
age.
The image must be a single-channel image.
It uses the following equation:
  w-1
 yproj[y] = SUM img[x][y][0]
  x=0
where y = 0, 1, ..., h - 1.
The function takes the following arguments:
yproj Pointer to Y-projection vector, where length is equal to the number of rows in the source image (in other
words, the image height).
img Pointer to an input image.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
