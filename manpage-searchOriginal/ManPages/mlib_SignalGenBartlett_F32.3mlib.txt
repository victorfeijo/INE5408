mlib_SignalGenBartlett_F32 <-> Bartlett window generation
cc [ flag... ] file... -lmlib [ library... ]
#include <mlib.h>
mlib_status mlib_SignalGenBartlett_F32(mlib_f32 *window, mlib_s32 n);
The mlib_SignalGenBartlett_F32() function generates the normalized coefficients of the Bartlett window.
The function takes the following arguments:
window Generated window coefficient array.
n Length of window array.
The function returns MLIB_SUCCESS if successful. Otherwise it returns MLIB_FAILURE.
